# emojiSweeper
Classic Minesweeper game implemented in React. 

# Run it locally
Clone the repo into your local machine:
```terminal
git clone https://github.com/fac27/emojiSweeper
```
Go into the project directory:
```terminal
cd emojiSweeper
```
Install dependencies:
```terminal
npm i
```
Run local web server:
```terminal
npm run server
```
